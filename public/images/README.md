## Full-cell replacement images

These replace an entire combo/exercise cell when "freestyle" or "shoeshine" is entered.
Provide one pre-scaled version per column count:

| File | Used when |
|------|-----------|
| `freestyle_3col.png` | 3-combo blocks (Strength, Resistance, Endurance, Power) |
| `freestyle_4col.png` | 4-combo blocks (Force) |
| `freestyle_6col.png` | 6-combo blocks (Speed & Stamina) |
| `shoeshine_3col.png` | 3-combo blocks |
| `shoeshine_4col.png` | 4-combo blocks |
| `shoeshine_6col.png` | 6-combo blocks |

## Notes
- Supported formats: `.png`, `.jpg`, or `.svg` (tried in that order)
- Missing images are silently skipped — the app still works
- After adding/changing files, push to GitHub and Vercel will redeploy
