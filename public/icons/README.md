# Icons

Drop your image files here. The app will auto-load them at startup.

## Expected filenames

| File | Used for |
|------|----------|
| `duck.png` | Duck movement icon |
| `roll.png` | Roll movement icon |
| `dash.png` | Dash movement icon |
| `fire.png` | Fire/intensity icon |
| `freestyle.png` | Freestyle combo image |
| `shoeshine.png` | Shoeshine combo image |

## Supported formats
`.png`, `.jpg`, or `.svg` — the app tries each in that order.

## Notes
- Filenames must be lowercase and match exactly
- Any missing icons are silently skipped (won't break the app)
- After adding/changing files, redeploy on Vercel (push to GitHub)
